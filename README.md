# PULSEBS_group2


## Docker instructions

```bash
git clone https://github.com/se2-group2/PULSeBS.git
cd PULSeBS
sudo docker pull alessiodeioanni/pulsebs_group2:frontend
sudo docker pull alessiodeioanni/pulsebs_group2:backend
sudo docker-compose up
```
